# 🎄 Advent of Code 2022

Solutions for [Advent of Code](https://adventofcode.com/) in [Rust](https://www.rust-lang.org/).


## Usage

- Run a single day: `cargo run DAY_NUMBER`
- Run whole suit: `cargo run`
